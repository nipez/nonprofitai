# Roadmap

## Phase 0: Context

- Define mission and target beneficiaries.
- Clarify the first concrete deliverable.
- Gather existing assets, constraints, and stakeholder priorities.
- Identify success measures.

## Phase 1: Strategy

- Translate context into a project plan.
- Prioritize work by impact, urgency, and uncertainty.
- Decide what should be built, written, researched, or designed first.

## Phase 2: Build

- Implement the first deliverable.
- Validate with the intended audience or stakeholders.
- Iterate based on feedback.
- Deploy the project hub to Railway from GitHub.
- Add Supabase-backed persistence after the Barb/Nick workflow is validated.

## Phase 3: Operations

- Document repeatable workflows.
- Define ownership, maintenance, and governance.
- Track outcomes against success measures.
- Add auth, role boundaries, and data-retention policies before storing sensitive AMPT materials.
