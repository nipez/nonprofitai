# Static Deploy Package

Upload the contents of this folder to Cloudflare Pages, GitHub Pages, Netlify, or another static host.

## Routes

- / - public marketing site
- /hub/ - working platform prototype

## Cloudflare Pages

Use Direct Upload and upload this folder, or zip the folder contents and upload the ZIP. No build command is required.

## Notes

The prototype stores edits in browser localStorage. No backend or database is required for this static demo.
